# Actividad_Desarrollo_Web

## Crear un sitio web profesional, responsivo y desplegado, cuya nota final dependerá de la capacidad del estudiante para justificar sus decisiones de código ante el profesor.



calcularDescuento(monto): Esta función mira el precio que pusiste. Si es mucha plata (más de $70,000), multiplica por 0.15 para darte un descuento grande. Si es poco, multiplica por 0.05 para un descuento más chico.

pagarPermiso(): Es la que hace todo el trabajo de "oficina". Toma el número que escribiste, llama a la función de arriba para saber el descuento, hace la resta y te muestra el mensaje con el total a pagar.
---------------------------------------------------------------
actualizarLista(): Borra la lista vieja de la pantalla y dibuja la nueva para que todos vean quién sigue.

agregarNormal(): Toma el nombre que escribiste y lo pone al final de la fila.

agregarUrgencia(): Toma el nombre pero lo pone al principio de la fila.
---------------------------------------------------------------
buscarBeneficiario(dato): Revisa la lista nombre por nombre, si encuentra el que escribiste, dice "¡Aquí está!" si termina de buscar y no lo vedice "No está".

verificarBeneficiario(): Es la función que conecta el buscador con la pantalla. Recibe el nombre, le pregunta al detective si lo encontró y te muestra el mensaje de "Verificado" o "No registrado".
